Classpublic class Menu
InheritanceMenu List SelectableList BaseScrollPane UIComponent flash.display.Sprite
ImplementsIRendererContainer

The Menu class creates an array of menus from an XML data provider. Add the Menu to your library (either by dragging it onto the stage and then deleting it, or dragging it directly to your library). Then call the static Menu.createMenu(parent) method to create a menu, and call yourMenuInstance.show() to display it.

dataProviderproperty 
dataProvider:DataProvider  [read-write]

Gets or sets the data model of the list of items to be viewed. A data provider can be shared by multiple list-based components. Changes to the data provider are immediately available to all components that use it as a data source. To use XML data, you should set the dataProvider as type XMLDataProvider: menu.dataProvider = new XMLDataProvider(someXML);

You can also use an array of objects in the format: var dp:Array = [ { label:"top level item A", data: [ { label:"submenu A1" }, { label:"submenu A2" } ] }, { label:"top level item B" }, { label:"top level item C" } ]; menu.dataProvider = new DataProvider(dp);

However, using the createMenu() method and passing in XML or an array will do this for you.

The default value is null.

Method detail
createMenu()method
public static function createMenu(inParent:DisplayObjectContainer, xmlDataProvider:Object = null):Menu

Creates and returns an instance of the Menu class. The Menu control's content is determined by the method's xmlDataProvider argument. The Menu control is placed in the parent container specified by the method's parent argument. This method does not show the Menu control. Instead, this method just creates the Menu control and allows for modifications to be made to the Menu instance before the Menu is shown. To show the Menu, call the Menu.show() method.

Parameters
inParent:DisplayObjectContainer — A container that the PopUpManager uses to place the Menu control in. The Menu control may not actually be parented by this object.
 
xmlDataProvider:Object (default = null) — The data provider for the Menu control.

Returns
Menu — An instance of the Menu class.

show()method 
public function show(xShow:Object = null, yShow:Object = null):void

Shows the Menu control. If the Menu control is not visible, this method places the Menu in the upper-left corner of the stage, resizes the Menu control as needed, and makes it visible. The x and y arguments of the show() method specify the coordinates of the upper-left corner of the Menu control relative to the stage, which is not necessarily the direct parent of the Menu control. For example, if the Menu control is in a container, the x and y coordinates are relative to the stage, not the container.

Parameters
xShow:Object (default = null) — Horizontal location of the Menu control's upper-left corner (optional).
 
yShow:Object (default = null) — Vertical location of the Menu control's upper-left corner (optional).
